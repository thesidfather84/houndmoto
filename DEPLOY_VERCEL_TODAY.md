## 🆘 IF SOMETHING BREAKS

### **"I see an error"**

1. Go to Vercel dashboard
2. Click on "Deployments" tab
3. If latest is red (❌), previous is green (✅)
4. Click the green one → "Redeploy"
5. Wait 30 seconds
✅ Fixed

### **"Parts search not working"**

1. Check browser console (F12 → Console)
2. Any red errors?
3. Take screenshot
4. Go to GitHub
5. Create "Issue" with screenshot
6. I can help fix

### **"Running out of bandwidth"**

Don't panic! You've got 100GB/month free.

If hitting 80GB:
1. Go to Vercel dashboard
2. Click "Settings" → "Billing"
3. Click "Upgrade to Pro" ($20/month)
4. Add payment method
5. Done! Now you have 1TB/month
